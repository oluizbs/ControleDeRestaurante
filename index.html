Faça um aplicativo que controle de pedidos de um restaurante. Você deve montar um cardápio dinâmico que busque os pratos via AJAX. em https://rafaelescalfoni.github.io/desenv_web/restaurante/items.json No aplicativo de pedidos, o cliente pode selecionar um ou mais pratos e então os itens devem ir para uma lista de preparos e montar uma detalhamento da compra, com a quantidade comprada, preço unitário e total. Lembre-se de adicionar 10% de taxa de serviço. Seu app deve montar o cardápio via AJAX Seu app deve usar a Storage API para armazenar os itens comprados Seu app deve permitir o cancelamento de itens do pedido em 5 minutos (Desculpe mas seu pedido já foi enviado para a cozinha) Seu app deve fechar a conta (lembre-se de detalhar a compra para conferência, tipo do McDonalds, segundo a Dhiovana, adicionar 10% de taxa de serviço) Document
